# 0X0A. C- argc,argv
***
This Directory contains Tasks on ArgC and ArgV..
<br>
### ArgC - 
		This is also called Argument Count. It counts the number of arguments pass on the command line, It is also the size of the argv array.
		
### ArgV
		This is also called Argument Vetor. It is an array of size argc, it contains the program command line arguments.
